Safe Hamlet

safehamlet.com github.com/ser-dialest/final-project Show Description

Safe Hamlet is a turn-based strategy game built from the ground up in React. Its back end is written using Node.js and Express.js., and users are able to log in and save their progress via its connected MongoDB database. It's still in development, but please, spend a few minutes trying it out! In addition to both the front and back-end code, the design, art, and music were all created by me.

Siliconscrapera

warm-island-18313.herokuapp.com/ github.com/ser-dialest/siliconscrapera Show Description

Siliconscrapera scrapes headlines and summaries, as well as the authors and publication date, of articles from gaming news website, SiliconEra. It's layout is presented using Bootstrap and organized using handlebars. Users are able to scrape new articles from the site, get linked to full articles on the original site, and leave and delete comments stored in MongoDB.

Pokemon GIF Finder

jlheatherly.com/GifTastic/ github.com/ser-dialest/GifTastic Show Description

The Pokemon GIF Finder leverages the Giphy API to show animated images of Pokemon. Users can search using pre-existing buttons, or create buttons that will search for the Pokemon of their choice. Once images are shown, their animations can be stopped and started, and more images can be requested. Users can only search for Pokemon. (Supports all Pokemon before the recently released Sword and Shield, property of Creatures, Inc., The Pokemon Company, and Nintendo.)

FF Battle Royale

jlheatherly.com/unit-4-game/ github.com/ser-dialest/unit-4-game/ Show Description

An experiment in jQuery and responsive design, FF Battle Royale is a simple game using assets from the classic NES game Final Fantasy. The animation techniques used here allow for a completely faithful recreation of the original animations, and the layout is designed to maintain the pixel ratios of the source artwork at virtually any resolution. (Graphics and sound property of SquareEnix.)

Advance Wars RPS

jlheatherly.com/RPSMultiplayer/ github.com/ser-dialest/RPSMultiplayer Show Description

Advance Wars RPS is a real-time, on-line, multiplayer, rock paper scissors game written in jQuery. Players input a user name, which is stored in a real-time Firebase database. The user is not password protected, but is persistent in the database, which keeps track of wins and losses over time. The nature of the database is also how players are able to play against each other in real time. Please note that two players are required to play the game. (Two browsers can be used.) This project is functional, but is missing animations. (Graphics property of Intelligent Systems and Nintendo.)
