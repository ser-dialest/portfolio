Safe Hamlet

safehamlet.com github.com/ser-dialest/final-project Show Description

Safe Hamlet is a turn-based strategy game built from the ground up in React. Its back end is written using Node.js and Express.js., and users are able to log in and save their progress via its connected MongoDB database. It's still in development, but please, spend a few minutes trying it out! In addition to both the front and back-end code, the design, art, and music were all created by me.

Siliconscrapera

warm-island-18313.herokuapp.com/ github.com/ser-dialest/siliconscrapera Show Description

Siliconscrapera scrapes headlines and summaries, as well as the authors and publication date, of articles from gaming news website, SiliconEra. It's layout is presented using Bootstrap and organized using handlebars. Users are able to scrape new articles from the site, get linked to full articles on the original site, and leave and delete comments stored in MongoDB.

Pokemon GIF Finder

jlheatherly.com/GifTastic/ github.com/ser-dialest/GifTastic Show Description

The Pokemon GIF Finder leverages the Giphy API to show animated images of Pokemon. Users can search using pre-existing buttons, or create buttons that will search for the Pokemon of their choice. Once images are shown, their animations can be stopped and started, and more images can be requested. Users can only search for Pokemon. (Supports all Pokemon before the recently released Sword and Shield.)
